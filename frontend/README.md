# Frontend - Qurbani Management System

React.js Admin Dashboard for Qurbani Management

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Environment Variables

Configure `.env`:
- REACT_APP_API_URL=http://localhost:5000/api

## Features

- Admin authentication
- User management (CRUD)
- Group management with member limits
- Qurbani tracking and status updates
- Real-time dashboard with statistics
- Responsive design

## Components

- **Login**: Admin authentication
- **Dashboard**: Overview and statistics
- **Users**: Individual user management
- **Groups**: Group creation and member management
- **QurbaniList**: View and manage all Qurbani requests
- **Navbar**: Navigation and logout

## Tech Stack

- React.js 18
- React Router 6
- Axios
- CSS3
